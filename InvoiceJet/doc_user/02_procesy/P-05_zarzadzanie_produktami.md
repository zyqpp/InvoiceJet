# Jak zarządzać katalogiem produktów i usług

## Po co katalog produktów?
Dzięki niemu nie musisz za każdym razem ręcznie wpisywać nazwy, ceny i stawki VAT przy tworzeniu faktury. Wystarczy, że wybierzesz produkt z listy podpowiedzi.

## Jak dodać produkt lub usługę?

1. W menu bocznym kliknij **Products**
2. Kliknij **Add Product**
3. W oknie dialogowym wypełnij:

   | Pole | Co wpisać |
   |---|---|
   | **Name** | Nazwa produktu lub usługi (np. „Usługa programistyczna", „Laptop Dell") |
   | **Price** | Cena jednostkowa |
   | **Unit of Measurement** | Jednostka miary (np. szt., kg, godz., m²) |
   | **TVA Value** | Stawka VAT w procentach (np. 23) |
   | **Contains TVA** | Zaznacz, jeśli podana cena jest **brutto** (z VAT). Odznaczone = cena netto |

4. Kliknij **Save**

## Jak edytować produkt?
Kliknij na wiersz produktu w tabeli — otworzy się formularz edycji.

## Jak usunąć produkt?
1. Zaznacz produkt (checkbox po lewej)
2. Kliknij **Delete**

## Wskazówka
Usunięcie produktu z katalogu nie wpływa na już wystawione faktury — dane pozycji są zapisane bezpośrednio w dokumencie.
