# Jak skonfigurować dane firmy

## Kiedy to robisz?
Raz — zaraz po rejestracji. Bez tych danych nie możesz wystawiać faktur.
Możesz też wrócić w dowolnym momencie, żeby zaktualizować dane.

## Kroki

1. W menu bocznym kliknij **Firm Details**
2. Wypełnij formularz:

   | Pole | Co wpisać |
   |---|---|
   | **Firm Name** | Pełna, oficjalna nazwa Twojej firmy |
   | **CUI** | Numer identyfikacji podatkowej |
   | **Reg. Com.** | Numer rejestracji handlowej |
   | **Address** | Adres siedziby firmy |
   | **County** | Okręg / województwo |
   | **City** | Miasto |

3. Kliknij **Save**

## Wskazówka: autouzupełnianie z rejestru
Jeśli znasz numer CUI swojej firmy:
1. Wpisz go w polu **CUI**
2. Kliknij ikonę chmury ☁ obok tego pola
3. Aplikacja automatycznie pobierze i wypełni pozostałe dane z publicznego rejestru

## Co dalej?
Po uzupełnieniu danych firmy skonfiguruj konto bankowe i serie numeracji, a następnie możesz wystawiać faktury.
