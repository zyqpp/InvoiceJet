# Jak skonfigurować serie dokumentów (numeracja)

## Po co seria dokumentów?
Seria decyduje o tym, jak będą numerowane Twoje faktury — np. „FV/2024/001", „FV/2024/002". Bez skonfigurowanej serii nie możesz wystawić faktury.

## Jak dodać serię?

1. W menu bocznym kliknij **Document Series**
2. Kliknij **Add Document Series**
3. Wypełnij formularz:

   | Pole | Co wpisać |
   |---|---|
   | **Document Type** | Typ dokumentu: **Faktura**, **Proforma** lub **Storno** |
   | **Series Name** | Prefiks numeru (np. „FV/2024/", „PRO/") |
   | **First Number** | Numer, od którego zaczniesz (zwykle `1`) |

4. Kliknij **Save**

## Przykład
Chcesz numerować faktury jako „FV/2024/001", „FV/2024/002" itd.:
- Document Type: Faktura
- Series Name: `FV/2024/`
- First Number: `1`

## Jak edytować serię?
Kliknij na wiersz serii w tabeli.

## Jak usunąć serię?
Zaznacz (checkbox) i kliknij **Delete**.

## Ważne informacje
- Zalecane jest skonfigurowanie oddzielnej serii dla każdego typu dokumentu
- Możesz mieć kilka serii dla tego samego typu (np. różne lata)
- Numer bieżący jest aktualizowany automatycznie przy każdym nowym dokumencie — nie musisz go pilnować
