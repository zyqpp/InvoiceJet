# P-05b — Jak skonfigurować serie dokumentów (numeracja)

## Po co?
Seria decyduje o tym, jak będą numerowane Twoje faktury — np. „FV/2024/001", „FV/2024/002".

> ⚠️ Bez skonfigurowanej serii nie możesz wystawić faktury.

---

## Jak dodać serię?

1. W [menu bocznym](../01_ekrany/03_nawigacja.md) kliknij **Document Series** → [opis ekranu](../01_ekrany/09_serie_dokumentow.md)
2. Kliknij **Add Document Series**
3. Wypełnij formularz:

   | Pole | Co wpisać |
   |---|---|
   | **Document Type** | Typ: [Faktura](../01_ekrany/10_faktury.md), [Proforma](../01_ekrany/11_proformy.md) lub [Storno](../01_ekrany/12_storna.md) |
   | **Series Name** | Prefiks numeru (np. „FV/2024/", „PRO/") |
   | **First Number** | Numer startowy (zwykle `1`) |

4. Kliknij **Save** ✅

---

## Przykład
Chcesz numerować faktury jako „FV/2024/001", „FV/2024/002"…:
- Document Type: Faktura
- Series Name: `FV/2024/`
- First Number: `1`

---

## Jak edytować serię?
Kliknij na wiersz serii w tabeli → formularz edycji.

## Jak usunąć serię?
Zaznacz (☐) → **Delete**.

---

## Ważne informacje
- Zalecane: osobna seria dla każdego typu dokumentu
- Numer bieżący rośnie automatycznie przy każdym nowym dokumencie
