# Nawigacja — pasek górny i menu boczne

## Pasek górny (navbar)

Widoczny na każdym ekranie po zalogowaniu. Zawiera:

| Element | Opis |
|---|---|
| **Przycisk menu** | Otwiera lub zamyka boczne menu nawigacji (sidebar) |
| **Przełącznik motywu** | Zmienia wygląd aplikacji między jasnym a ciemnym trybem |
| **Przycisk profilu** | Okrągły przycisk z inicjałami — kliknij, aby się **wylogować** |

## Menu boczne (sidebar)

Drzewo nawigacji po lewej stronie ekranu. Prowadzi do wszystkich sekcji aplikacji:

| Sekcja | Do czego prowadzi |
|---|---|
| **Dashboard** | Pulpit główny ze statystykami |
| **Firm Details** | Dane Twojej firmy |
| **Clients** | Lista kontrahentów (klientów) |
| **Bank Accounts** | Konta bankowe firmy |
| **Products** | Katalog produktów i usług |
| **Document Series** | Serie i autonumeracja dokumentów |
| **Invoices** | Lista wystawionych faktur |
| **Invoice Proformas** | Lista proform |
| **Invoice Stornos** | Lista faktur korygujących |

Aktualnie wybrana sekcja jest wyróżniona w menu pogrubieniem i jaśniejszym tłem.

## Wygasła sesja

Jeśli przez dłuższy czas nie korzystałeś z aplikacji, pojawi się okno z informacją o wygaśnięciu sesji. Kliknij przycisk, aby wrócić do ekranu logowania i zalogować się ponownie.
