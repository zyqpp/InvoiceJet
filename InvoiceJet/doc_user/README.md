# InvoiceJet — Dokumentacja użytkownika

Witaj w dokumentacji użytkownika aplikacji **InvoiceJet** — systemu do wystawiania faktur i zarządzania dokumentami sprzedaży.

## Struktura tej dokumentacji

| Folder / Plik | Zawartość |
|---|---|
| `01_ekrany/` | Opis każdego ekranu aplikacji — co widać, co można zrobić |
| `02_procesy/` | Instrukcje krok po kroku — jak wykonać konkretne zadanie |
| `zrodla.md` | Wykaz dokumentów źródłowych, na których oparto tę wiedzę |
