# register — Ekran rejestracji

Ekran rejestracji nowego konta w aplikacji InvoiceJet. Publicznie dostępny — nie wymaga autoryzacji. Po rejestracji token JWT jest natychmiast zapisywany i użytkownik przekierowywany do dashboardu bez dodatkowego kroku logowania.

## Drzewo zawartości

```
register/
├── README.md    ← Ten plik
└── ekran.md     ← RegisterComponent — formularz rejestracji
```

## Kluczowe dokumenty

- `ekran.md` — pełna dokumentacja ekranu rejestracji: pola, walidacja, przepływ, wywołania API

## Powiązane katalogi

- `../login/` — ekran logowania (link „Zaloguj się")
- `../dashboard/` — cel nawigacji po pomyślnej rejestracji
