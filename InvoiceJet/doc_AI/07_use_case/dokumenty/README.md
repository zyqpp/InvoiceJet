# dokumenty — Przypadki użycia zarządzania dokumentami

Przypadki użycia dotyczące wystawiania i zarządzania wszystkimi typami dokumentów finansowych: faktur, faktur proforma i faktur storno. Obejmuje również przeglądanie statystyk na dashboardzie.

## Pliki

| Plik | Opis |
|---|---|
| `uc_faktury.md` | Wystawianie, edycja i pobieranie PDF faktur (documentTypeId=1) |
| `uc_faktury_proforma.md` | Wystawianie i zarządzanie fakturami proforma (documentTypeId=2) |
| `uc_faktury_storno.md` | Konwersja dokumentów na storno i zarządzanie stornami (documentTypeId=3) |
| `uc_dashboard.md` | Przeglądanie statystyk firmy i wykresu przychodów na dashboardzie |
