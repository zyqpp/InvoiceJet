# Document Series - Elementy elementarne

| Element | Typ | Komponent | Opis |
|---|---|---|---|
| Lista serii dokumentow | ekran | `DocumentSeriesComponent` | Grid serii aktywnej firmy. |
| Dialog serii | dialog | `AddOrEditDocumentSeriesDialogComponent` | Dodanie albo edycja serii. |
| Usuwanie | operacja masowa | `deleteSelected()` | Usuwa zaznaczone serie. |
