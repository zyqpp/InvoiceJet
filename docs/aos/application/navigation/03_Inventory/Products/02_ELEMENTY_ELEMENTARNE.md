# Products - Elementy elementarne

| Element | Typ | Komponent | Opis |
|---|---|---|---|
| Lista produktow | ekran | `ProductsComponent` | Grid produktow aktywnej firmy. |
| Dialog produktu | dialog | `AddOrEditProductDialogComponent` | Dodanie albo edycja produktu. |
| Usuwanie | operacja masowa | `deleteSelected()` | Usuwa zaznaczone produkty. |
